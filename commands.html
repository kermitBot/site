Kermit BOT

A multi-tool Discord robot



Available Commands

?play [song name/url] Plays the song you input
?skip Skip the current song to the next one
?pause Pauses the music playback
?resume Resumes the music playback
?stop Stops the music playback
?queue Displays the song titles in the queue

Fun Commands

?meme Displays a random meme from a subreddit
?ping Show bot stats (for nerds)

Admin Commands

Only roles with the "ADMINISTRATOR" permission can access these commands

?poll [text] Creates a poll
?clear [amount] Delete messages
?votekick [@user] Creates a user-based votekick
?say [text] Talk as the bot
?tempmute [1s/1m/1h/1d/1y] Temporarely mute someone
